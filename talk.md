## *Closure*
<br>
@@@ ruby

    user = "Foo Bar"
    10.times do
      puts user
    end

@@@
<br>
<br>
##### donde *user* es una variable que fue creada fuera del bloque y el bloque puede acceder a ella

!SLIDE

## *Lambda*
<br>
@@@ ruby

    reverse_string_order = -> x, y { y.to_s <=> x.to_s }
    [1, 20, 3].sort &reverse_string_order

    square = -> n { n * n }
    [1, 2, 3, 4, 5].map &square

@@@
